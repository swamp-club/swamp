# swamp

Deno-based CLI for AI Native Automation.

## Installation

```bash
deno install -A --name swamp main.ts
```

## Usage

```bash
# Run the CLI
deno run dev

# Show version
swamp version
```

## Development

```bash
# Run tests
deno run test

# Type check
deno check

# Lint
deno lint

# Format
deno fmt
```
